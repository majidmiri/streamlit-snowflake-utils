# Snowflake Data Lineage Web App (using Streamlit)

This repository contains a Streamapp web app for producing lineage graphs for a specific object.


## Setup Instructions

- Make sure all libraries specified in `requirements.txt` are installed.
- Rename `creds_template.json` to `creds.json` and add your Snowflake account's connection details.
- Run the application using the following command:
```
streamlit run app.py
```
